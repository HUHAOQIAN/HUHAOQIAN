## About

I build practical tools that make development and on-chain operations easier to run, repeat, and maintain.

- Working across **TypeScript**, **Python**, **Shell**, **Go**, and **Rust**.
- Interested in **CLI tooling**, **macOS terminal setup**, **Web3 automation**, **Bitcoin ecosystem experiments**, and **agent workflows**.
- Prefer small, direct utilities that solve real operational problems.
